# MUXI Repositories

All MUXI repositories live under the [muxi-ai](https://github.com/muxi-ai) organization.


## Core

| Repository | Description |
|------------|-------------|
| [server](https://github.com/muxi-ai/server) | Agent orchestration platform |
| [runtime](https://github.com/muxi-ai/runtime) | Formation execution environment |
| [cli](https://github.com/muxi-ai/cli) | Command-line interface |
| [registry](https://github.com/muxi-ai/registry) | Formation distribution hub |
